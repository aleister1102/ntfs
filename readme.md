# NTFS 
Chương trình đọc phân vùng NTFS.

Các câu lệnh:
- `dir`: liệt kê các tập tin/thư mục có trong thư mục hiện tại.
- `cd`: thay đổi thư mục.
    - `cd ..`: về thư mục cha.
    - `cd <input>`: vào thư mục input.
- `cat <input>`: xem nội dung của tập tin input.
- `cls`: xóa console.
- `exit`: thoát chương trình.
